# VilchesApp – Demo

Live demo av VilchesApp, ett komplett ordersystem för inredning & montage.

## Vad demo:n visar

| Skärm | Beskrivning |
|-------|-------------|
| **Inloggning** | Välj roll (Admin eller Montör) för att se respektive vy |
| **Dashboard** | Intäktsstatistik, aktiva projekt med progress, senaste ordrar |
| **Ordrar** | 8-stegs orderflöde från skannad papperorder till slutförd installation |
| **Projekt** | Projektkort med progress, team och leveransdata |
| **Offerter** | AI-genererade offerter med konverteringsstatistik |

## Fullständiga systemet erbjuder

- OCR-skanning av papperordrar (AJ Produkter, Lekolar, Kinnarps m.fl.)
- 7-stegs orderstatus med realtidsspårning
- Projektledning med teamtilldelning och kartvy
- AI-driven offertgenerering baserad på historiska projekt
- Tidrapportering med attestflöde och Fortnox-integration
- Fakturering med ROT/RUT-avdrag
- Roller: Admin, Montör, Anställd, Revisor

## Teknisk stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- React Router v6
- Recharts
